# reactive-blog

## Requirements

* Tested on node 5+
* npm install -g bower
* Elasticsearch 2+
* npm install -g webpack

## Install

```bash
npm install
l10ns compile # compile localizations
./node_modules/.bin/webpack --colors --progress # or add --watch for file-watching
```

or

```bash
npm install; npm run build;
```
## Run

```bash
npm start
```
